# Checkerboard

A simple computer simulation of a checker game

## Getting Started

The application is written in python with a few dependencies 

### Dependancies


```
tkinter(for gui)
pygame(for audio)
```

### Installing

```
pip install tkinter
pip install pygame
```
## Running

To run the application:

```
python checkerboard.py
```
